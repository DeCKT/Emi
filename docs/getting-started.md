# Getting Started

Welcome to Emi. This quick guide will help you create your first event and send invitations in just a few steps.

## Create Your Account

## Create an Event

From the Dashboard, click Create Event and enter basic details like the event name, date, time, and location. Save your event to continue.

## Send Invitations

Open your event and go to the Invitations tab. Add guests, then send invitations directly or copy their unique links to share.

## Track Responses

As guests RSVP, their status appears immediately. You can resend invites, follow up, or message guests as needed.

![An Image](../image.jpg)

## Update Your Event

If you change event details, Emi can automatically notify your guests so everyone stays informed.
